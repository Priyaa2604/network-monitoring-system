
# Network Monitoring & Alerting System

A Python-based system to monitor website availability and performance.

## Features
- Website uptime monitoring
- Response time tracking
- Failure detection
- Real-time alerts

## Tech Stack
- Python
- Requests library

## Outcome
Improved understanding of network reliability, monitoring, and automation.
## How It Works
- Sends HTTP requests to check website availability
- Measures response time (latency)
- Detects downtime and prints alerts

## Sample Output
[UP] google.com | 120 ms  
[DOWN] example.com
